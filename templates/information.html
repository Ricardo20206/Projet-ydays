ℹ️ Informations

À propos de l'application

Cette application Flask vous permet de gérer et traiter vos médias (vidéos et images) de manière simple et efficace. Vous pouvez uploader, visualiser, et envoyer vos fichiers à une API de traitement pour obtenir des résultats optimisés.

Fonctionnalités

Gestion des vidéos : Upload, visualisation et traitement de vidéos

🖼️ Gestion des images : Upload, visualisation et traitement d'images

🔍 Recherche : Barre de recherche pour trouver rapidement vos médias

Traitement API : Envoi de vos médias à une API externe pour traitement

Formats supportés

Vidéos : MP4, AVI, MOV, MKV, WEBM

Images : JPG, JPEG, PNG, GIF, BMP, WEBP, SVG

Comment utiliser

1. Sélectionnez une page dans le menu (Vidéo ou Image)

2. Cliquez sur "Charger" et sélectionnez votre fichier

3. Visualisez votre média dans la zone d'affichage

4. Optionnellement, envoyez-le à l'API de traitement

5. Téléchargez le résultat traité

Technologies utilisées

• Flask - Framework web Python

• HTML5 / CSS3 - Interface utilisateur

• JavaScript - Interactivité

• API REST - Traitement externe
